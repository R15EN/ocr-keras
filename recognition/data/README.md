
#### The structure of the directory
```
gt.txt
images/
    0/
        0.jpg
        1.jpg
        ...
        9998.jpg
        9999.jpg
    1/
    ...
```

The format of `gt.txt` is as follows. Image path and label are separated by tab. (`<image_path>\t<label>`)


```
images/0/0.jpg	10
images/0/1.jpg	date:
...
```
